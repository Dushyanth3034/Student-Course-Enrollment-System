# Student Course Enrollment System

## 📌 Description
This is a SQL mini project that manages:
- Students
- Courses
- Enrollment
- Exams
- Results

## 🛠️ Features
- Table creation
- Sample data insertion
- Trigger to prevent duplicate exams
- Stored Procedure for student report
- Function for grade calculation
- View for enrollment data

## ▶️ How to Run
1. Open MySQL Workbench
2. Run student_project.sql
3. Execute:
   CALL get_student_report(1);

## 📊 Output
- View student enrollment data
- Get student marks and grades
